## Getting Started

### Clone the repo

[github](https://github.com/brito-ca/brito-website.git)

### Install

```bash
npm install
```

### Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
